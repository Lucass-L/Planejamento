

PLANEJAMENTO ESCOLAR

Criar um aplicação que permita o professor ter seu login unico e visualizar os repasses da diretoria; 
Neste projeto teremos 3 usuarios com opções diferentes: Diretoria, professor, visitante;

    Diretoria:
    Terá que ter acesso a visualização de todas as turmas, disciplinas e professores!
        Nova aba que vai permitir:

     Poder criar novos cards para atrubuir a um professor ou  fazer a transferência de matérias, gerenciar a senha e username de cada professor.

    Professor: 
    Terá acesso apenas as turmas relacionadas e a disiciplina atuante em cada uma delas, conseguira vizualizar de forma fácil um card com um link para o google docs onde deverá registrar o planejamento! Deve ter um botao em cada card para registrar alterações [finalizado], registrando assim o horário em que foi feita a mudança para a diretoria ter facilidade na hora de conferir os projetos.

    Visitante: Aplicação Futura onde os estudantes (jovem protagonistas EMTI) devem acompanhar os planejamentos de cada professor mas não devem ter acesso à nada e nem permitir modificações.
